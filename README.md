# TG Parsing Bot

Бот для парсинга участников Telegram-групп с выгрузкой в Excel.

## Что умеет

- Парсить участников через `/parse`:
  - выбор чата из списка «Мои чаты»
  - или invite-ссылка (`https://t.me/+...`, `https://t.me/joinchat/...`)
- Выгружать результат в `.xlsx`
- Хранить историю запусков (`/history`)
- Показывать активный Telethon-аккаунт (`/account`)
- Держать постоянное нижнее меню (`/menu`)

## Важное про ограничения Telegram

Это ключевой момент для корректных ожиданий:

1. Полный список участников доступен не всегда.
2. В некоторых группах Telegram отдает только частичный список (часто админы/видимые участники).
3. В каналах (broadcast) полный список подписчиков обычно недоступен.
4. Если заявка в чат не одобрена, парсинг участников невозможен.

Если в чате 13k участников, а бот выдал 10, это обычно ограничение Telegram/прав доступа, а не поломка бота.

## Требования

- Python 3.10+
- Telegram-бот (`BOT_TOKEN`)
- Telegram API credentials (`API_ID`, `API_HASH`)
- Номер телефона аккаунта для Telethon (`PHONE`)

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Настройка `.env`

Скопируй шаблон:

```bash
cp .env.example .env
```

Заполни:

```env
BOT_TOKEN=123456:ABC...
API_ID=123456
API_HASH=xxxxxxxxxxxxxxxx
PHONE=+79990000000
```

## Запуск

```bash
python3 -m bot.main
```

## Первый вход Telethon

При первом использовании `/parse` бот попросит код входа прямо в Telegram-чате (не в консоли).

Если включен 2FA, бот попросит пароль вторым шагом.

## Смена активного аккаунта (номер в `.env`)

Если поменял `PHONE` в `.env`:

1. Перезапусти бота.
2. Запусти `/parse`.
3. Введи код для нового номера.

Сессии хранятся отдельно по номеру, поэтому аккаунты не смешиваются.

## Безопасность данных

Реализовано в проекте:

1. Шифрование Telethon-сессии (`.session.enc`).
2. Ключ шифрования хранится отдельно (`sessions/.session.key`).
3. Права доступа к сессиям ограничиваются (где ОС позволяет).
4. Чувствительные данные в логах маскируются (телефоны/токены/коды/пароли).

## Команды

- `/start` — старт и краткое описание
- `/parse` — запустить парсинг
- `/account` — активный Telethon-аккаунт
- `/history` — история выгрузок
- `/help` — инструкция в боте
- `/menu` — заново открыть постоянное меню

## Как пользоваться `/parse`

### Вариант 1 (рекомендуется)

1. `/parse`
2. Нажми «Выбрать из моих чатов»
3. Выбери чат кнопкой
4. Дождись Excel-файла

### Вариант 2

1. `/parse`
2. Отправь invite-ссылку:
   - `https://t.me/+...`
   - `https://t.me/joinchat/...`
3. Дождись результата

## Почему бывают неполные результаты

Чаще всего:

1. В группе скрыт список участников.
2. У аккаунта недостаточно прав для просмотра всех участников.
3. Telegram вернул частичный список по своим лимитам.
4. Это канал, а не группа.

## Типовые ошибки и что делать

### «Заявка отправлена и ожидает одобрения»

Ожидай подтверждение от админа чата, потом повтори `/parse`.

### `FROZEN_METHOD_INVALID`

Telegram временно ограничил действие для аккаунта.

Что делать:
1. Подожди и повтори позже.
2. Старайся не делать частые массовые запросы.
3. Используй стабильную сеть.

### Timeout/ConnectionError в логах

Это временные сетевые проблемы до Telegram.

Что помогает:
1. Стабильный интернет
2. Стабильный DNS
3. Повтор через несколько минут
